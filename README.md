Project 2 - Assembly Double Program

CMSC313 - Dr. Kidd

Reads a number from stdin, doubles it, and prints the result.

How to Run on gl-server

Compile: gcc -nostdlib -no-pie double.s -o double

Run: ./double

An empty line will appear. Type your number and press Enter. LIke 10 for example

Expected output: The double is: 20
